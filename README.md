# 🍔 Sky Take-Out System (Spring Boot + Vue)

A full-stack food ordering system built with **Spring Boot and Vue**, featuring **Redis caching**, **WebSocket real-time updates**, and **JWT-based authentication**.

## 🚀 Features

### 👤 User Side
- Browse dishes and categories  
- Add items to shopping cart  
- Place and manage orders  
- Online payment simulation  
- Order history tracking  

### 🛠 Admin Side
- Manage dishes and categories  
- Order management (accept, reject, dispatch)  
- Employee management  
- Business data statistics dashboard  

### ⚡ System Features
- Redis caching for performance optimization  
- WebSocket for real-time order notifications  
- JWT-based authentication and authorization  
- RESTful API design  
- Layered architecture (Controller / Service / Mapper)
